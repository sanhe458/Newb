# Newb Shader — 末日分支

> 基于 [Newb Shader 三河修改版](https://github.com/sanhe458/Newb) 的 Minecraft 基岩版末日废土风格光影分支。将主世界改造为昏暗天空、灰暗雾气滤镜与废土色调；云层仅做色调调整，形状/大小/速度/透明度等保持原样不变。

**当前版本**：`v2.0.0` ｜ **最低引擎版本**：`1.16.100` ｜ **协议**：MIT License

---

## 特性一览

| 模块 | 调整内容 |
| --- | --- |
| **色彩校正** | ACES 色调映射，曝光 0.82、对比度 0.90、饱和度 0.42，启用暖灰废土 TINT `vec3(0.94,0.88,0.80)` |
| **天空** | 阴灰天顶 `vec3(0.16,0.18,0.21)`、浑浊棕地平线、低日间通透度 0.14、水下浑浊灰绿 |
| **云层** | 平滑噪声云形状/尺寸/深度/速度/阴影/极光全部保持原样，仅叠加废土灰棕色调 `cloud_tint vec3(0.72,0.69,0.63)` |
| **光照** | 压暗日光（晨/午/夜均偏废土灰棕），日光强度 1.9，火把保留原暖色作为末日温暖光源对比 |
| **水体** | 海水/淡水/沼泽/水下统一改为浑浊灰绿废土色，保留水面波动与云层倒影 |
| **水下** | 水下波动与焦散不变，水下光照色改为浑浊灰绿 `vec3(0.18,0.28,0.26)` |
| **植被动效** | 树叶摆动、灯笼摇晃、雨天风力全部保持原样 |
| **雾与薄雾** | 薄雾密度提升至 0.30，空气雾 `#5C5650`、天气雾 `#47433D`、水雾 `#3A4A40` 统一灰棕浑浊 |

## 与原三河修改版的区别

- **整体压暗去饱和**：曝光 1.15→0.82，饱和度 1.25→0.42，营造昏暗末日滤镜
- **启用暖灰 TINT**：取消注释并设为 `vec3(0.94,0.88,0.80)`，赋予尘土废土色调
- **天空阴沉化**：天顶由蓝改为阴灰，地平线由日出橙改为浑浊棕，日间通透度大幅降低
- **浓雾灰棕化**：薄雾密度翻倍以上，空气/天气/水雾色统一改为灰棕浑浊色调
- **废土日光与水体**：日光晨/午/夜均压暗偏灰棕，海水/淡水/沼泽/水下统一浑浊灰绿
- **云仅色调调整**：新增 `cloud_tint` 常量对云色做纯乘法，不触碰任何形状/动态属性
- **末地与火把暖光保持原样**：末地本身已是暗紫废土风，火把暖色作为末日人造光源对比

## 通用性说明

本分支配色为**中性废土调色板**（灰棕暖灰为主、低饱和、阴灰天顶+暖棕地平线），不绑定特定地貌，可同时贴切多种末日场景：

- **荒野废土**（如惊变 100 天）：暖灰棕 TINT 与浑浊水体与枯草干土融合，浓雾压远符合尸潮来临前的压抑天色
- **末日城市**（如亡者世界）：阴灰天顶 + 灰棕雾与混凝土废墟、玻璃锈蚀同调，低饱和去除城市鲜艳残留感

其"冷上暖下"的天空—地平线渐变符合真实废土大气（地面扬尘暖、高空尘埃散射冷）的通用特征。

## 安装

1. 下载本仓库（`Code → Download ZIP`）或克隆：
   ```bash
   git clone https://github.com/sanhe458/Newb.git
   ```
2. 将整个文件夹放入基岩版资源包目录：
   ```
   games/com.mojang/resource_packs/
   ```
3. 启动游戏，在 `设置 → 全局资源` 中启用「Newb Shader 末日分支」。

> 也可将文件夹重命名为 `.mcpack` 后双击导入。

## 自定义配置

所有用户可调参数集中在 [`shaders/glsl/config_utilities/`](shaders/glsl/config_utilities) 目录下，每个文件顶部以注释标明参数含义与取值范围：

| 文件 | 控制内容 |
| --- | --- |
| `global.h` | 时间翻转、光照强度、雾与薄雾 |
| `color.h` | 色调映射、曝光、对比度、饱和度、TINT 色调 |
| `sky.h` | 天空、天顶、地平线、水下雾色 |
| `clouds.h` | 云层参数、极光效果、云色调 cloud_tint |
| `terrain.h` | 地形光照、软阴影、波浪、水体、水下 |
| `extra.h` | 日月旋转、发光纹理、水纹细节 |

> 修改 `// Code starts here` 注释以下的内容可能导致渲染异常，请仅调整上方注释标记的参数。

更多自定义说明参见 [原项目文档](https://devendrn.github.io/newb-shader/customization.html)。

## 目录结构

```
.
├── shaders/glsl/          # GLSL 着色器源码
│   ├── config_utilities/  # 用户可调参数（首选项）
│   └── utilities/         # 噪声、检测等内部工具函数
├── materials/             # 材质定义
├── textures/              # 贴图资源
├── fogs/                  # 雾设置
├── biomes_client.json     # 生物群系水体颜色
└── manifest.json          # 资源包清单
```
