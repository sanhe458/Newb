# Newb Shader — 三河修改版

一个基于 [Newb Shader](https://github.com/devendrn/newb-shader-mcbe) 的 Minecraft 基岩版光影包，在原版风格基础上做了一些调整和优化。

## 与原版的区别

- 着色器微调，视觉效果有所优化
- 部分着色参数重新调整
- 适配个人使用习惯的细节修改

## 安装

1. 下载此仓库的资源包
2. 将 `.mcpack` 或解压后的文件夹放入 `games/com.mojang/resource_packs/`
3. 在游戏设置中启用资源包
